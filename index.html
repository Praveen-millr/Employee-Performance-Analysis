Employee Performance Analysis

Project Overview

This project evaluates employee performance across teams and projects using a structured data analysis pipeline.

  • Dataset: Employee work logs, performance scores
  • Tools Used: Python (Pandas, Matplotlib), SQL, Excel
  • Goal: Identify high-performing employees and optimize resource allocation

Exploratory Data Analysis (EDA)

Analyzed key performance metrics:

  • Average hours worked per team.
  • Project-wise employee performance.
  • Identified high-performing teams.

Conclusion

The analysis highlights key insights:

  • Teams with higher collaboration tend to have better performance scores.
  • Some projects required overtime, leading to burnout risks.
  • Performance optimization strategies should focus on balancing workloads.
