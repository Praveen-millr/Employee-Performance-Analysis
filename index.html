Employee Performance Analysis

Project Overview

This project analyzes employee performance across multiple teams and projects. It follows a structured data analysis pipeline:

  • Step 1: Data Extraction - Gathering raw data from various sources.
  • Step 2: Data Cleaning - Handling missing values, removing duplicates.
  • Step 3: Data Processing - Merging datasets and transforming data.
  • Step 4: Exploratory Data Analysis (EDA) - Identifying key trends.
  • Step 5: Data Visualization - Generating charts to present insights.
